🎯 Getting Started

ClaraVerse requires a container engine to run AI models and services securely in isolated environments.

🔒 Why Containers?

Containers provide security, consistency, and easy management of complex AI dependencies.

🚀 Quick Setup

Recommended: Install Podman (more secure, no root required)

• macOS: brew install podman

• Linux: sudo apt install podman or sudo dnf install podman

• Windows: Download from podman-desktop.io


Alternative: Install Docker Desktop

• Download from docker.com
